# Space Launch Simulator

Java console application for a B1 course project.

The final goal is to configure a rocket, choose a mission, simulate a launch, calculate the result and cost, then save launch history.

## Requirements

- Java 17 or later
- Git

## Compile

```bash
javac -d out $(find src/main/java -name "*.java")
```

## Run

```bash
java -cp out Main
```

Expected output for the initial setup:

```text
Space Launch Simulator is ready.
```
